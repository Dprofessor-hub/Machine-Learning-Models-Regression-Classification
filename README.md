# 📌 Project Overview

This repository contains multiple supervised machine learning models developed using Python and Scikit-learn. The project demonstrates regression and classification techniques on real-world datasets, including salary prediction, lifestyle analysis, and breast cancer diagnosis.

The objective is to understand data preprocessing, model building, evaluation, and prediction using different machine learning algorithms.

---

# 🎯 Project Objectives

- Perform data preprocessing and cleaning
- Build regression models for prediction tasks
- Build classification models for medical diagnosis
- Compare different machine learning algorithms
- Evaluate model performance using appropriate metrics
- Demonstrate practical implementation of supervised learning
